# Real Estate Management System

## Project Description
This is a simple console-based Real Estate Management System written in Python. It allows users to add, view, search, update, and delete property listings. The application manages property details like title, location, price, size, and availability status.

## Features
- Add new properties with details such as title, location, price, and size.
- List all available properties with full information.
- Search properties by location keyword.
- Search properties by maximum budget.
- Update the status of a property (available, sold, rented).
- Delete properties from the system.
- Menu-driven user interface operating via the console.

## Technologies Used
- Python 3.x
